# SymQuPS

Symbolic implementation of the quantum phase space formulation of quantum mechanics utilizing SymPy.

(Formerly `moyalstar`: https://github.com/hendry24/moyalstar)

### Installation

You can install the package via `pip` or Github:

```
pip install symqups
```

or

```
pip install git+https://github.com/hendry24/SymQuPS
```
