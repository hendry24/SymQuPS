Welcome to SymQuPS!
===================

This is a package to algebraically go between the Hilbert space equipped with the canonical commutator
and the phase space equipped with the Poisson bracket structure, mainly used as a tool for the phase
space formulation of quantum mechanics, and quantization, by extension.
